MVP 页面信息架构
  1. 首屏，今日最强异动，突出一条主信号 + 异动榜
  2. Wall Shift 压住变化榜，做成你的招牌功能
  3. Smart Money 大额 / 新钱包 / 小众市场放量提醒
  4. 热门金融主题，帮助用户快速抓主线
  5. 每日播报区，适合后续接 Telegram / X 自动推送
  6. 专题入口，利率 / BTC / 大选 / 黄金 / 原油 / 地缘风险
第一阶段，静态原型 + 文案 + 结构
先把页面气质、模块、中文表达和“吸引人”做出来。
第二阶段，接 Gamma + CLOB 轮询
先做实时榜单、价格变化、成交活跃度和基础盘口快照。
第三阶段，接 WebSocket + 异动引擎
压单后撤、大额成交、地址异动、跨平台价差会在这时上线。
第四阶段，自动播报与传播
把最有戏的信号生成简洁中文播报，用于频道、社媒和日报。
这版我先按“最吸引人、最适合金融用户、最容易后续落地”为目标做成首页原型。下一步最值的是把这个页面接成真实数据版，并拆一个单独的“事件详情页 / 专题页”模板出来。
