核心判断
如果你想找韩国 DRAM / 存储链映射，KOSPI 200 本身已经很重，不只是“宽基”
很多人会把 KOSPI 200 先当成普通韩国大盘指数，但从这份 factsheet 看，它对三星电子和 SK 海力士的依赖度非常高。结合你给的 Roundhill Memory 持仓截图，里面三星电子 + SK 海力士合计约 37.62%，而 KOSPI 200 里这两家合计达到 61.63%。也就是说，KOSPI 200 里三星 + 海力士的占比，不只是大于一般人对 DRAM 主题的直觉，甚至显著高于这个专门做存储链映射的 DRAM 主题产品里两者的合计权重。

怎么用这个页面看 KOSPI 200
  • 如果你的目的是看韩国大盘风险偏好，这个指数当然有效。
  • 但如果你的目的是找 韩国 DRAM / 存储景气代理，它同样非常有用，因为三星 + 海力士在里面过于重。
  • 如果你担心“宽基不够纯”，这页正好说明：KOSPI 200 并不稀释 DRAM 链影响，反而把它放大到了指数级别。
  • 甚至从你给的 Roundhill Memory 截图对比看，KOSPI 200 里三星 + 海力士的集中度还要更夸张，因此它并不是“DRAM 暴露弱”的宽基，反而是一个对韩国存储龙头极其敏感的大盘指数。
  • 真正要注意的不是有没有 DRAM 暴露，而是它同时还叠加了韩国电子龙头、部分金融、汽车与工业权重，所以它是 以 DRAM / 电子为主导的蓝筹指数，不是纯题材篮子。
